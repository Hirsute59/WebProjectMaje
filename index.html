Bonjour,

Ce site internet est un projet étudiant pour en application le framework Bootstrap de Twiter.

Voici une petite définition d'un framework trouvé sur internet :

Les frameworks CSS sont spécialisés, comme leur nom l'indique, dans les CSS ! C'est-à-dire qu'ils nous aident à mettre en forme des pages web : organisation, aspect, animation...

Je vais maintenant vous présenter un tableau avec les avantage d'un tel framework et les désavantage :

Avantages Désavantages
Compatibilité avec tout les navigateurs Besoin de temps pour bien se servir du framework
Facilite grandement la partie Css d'un site Facile à utiliser

Vous pouvez aller dans la partie tutoriel pour avoir un apercu de la façon d'utiliser Bootstrap. Mais pour une aide plus approfondi, rendez-vous sur Bootstrap.
